# Visual Direction

## Objetivo

Definir una direccion visual consistente para los mockups en GIMP antes de programar.

## Principios visuales tomados del sitemap

- moderno
- limpio
- visualmente atractivo
- facil de navegar

## Lo que todavia debe decidirse

- personalidad de marca
- nivel de sobriedad vs expresividad
- paleta principal y paleta de apoyo
- tipografias
- estilo de fotografia o ilustracion
- estilo de iconografia
- uso de texturas, fondos o patrones
- estilo de botones, tarjetas y formularios

## Checklist para cada propuesta de diseno

- Se entiende la jerarquia visual en menos de 5 segundos.
- El CTA principal se distingue con claridad.
- La navegacion principal se siente simple.
- Hay consistencia entre home y paginas internas.
- El diseno funciona en desktop y mobile.
- El estilo visual apoya el tipo de contenido del sitio.

## Sistema minimo a definir en GIMP

- portada de Home
- encabezado y navegacion
- pie de pagina
- grid base
- espaciados principales
- botones
- tarjetas
- formularios
- banners o bloques destacados

## Referencias visuales a documentar

Por cada referencia que guardes, conviene anotar:

- que funciona
- que no funciona
- que elemento quieres adaptar
- en que pagina aplicaria
