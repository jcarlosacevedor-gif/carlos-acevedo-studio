# Project Decisions

## Uso del documento

Registrar decisiones para no perder contexto y separar lo resuelto de lo pendiente.

## Decisiones confirmadas

- El trabajo actual se concentra en estructura, organizacion y documentacion.
- El diseno se esta explorando primero en GIMP.
- No se debe generar codigo web todavia.
- El sitemap actual incluye: Home, About, Servicios, Blog, Contacto y Tienda.

## Suposiciones de trabajo

- El sitio combina objetivos informativos, comerciales y de contenido.
- Home funcionara como pagina de entrada y distribucion hacia las demas secciones.
- Servicios y Tienda son secciones de conversion.
- Blog apoyara descubrimiento, autoridad o visibilidad organica.

## Preguntas abiertas

- Cual es el objetivo principal del sitio: vender, captar leads, mostrar portafolio, informar o una mezcla.
- Quien es el usuario principal.
- Cual es la propuesta de valor principal.
- Que relacion exacta existe entre Servicios y Tienda.
- Si About debe ser institucional, personal o de marca.
- Si Blog tendra categorias definidas desde el inicio.
- Que tono visual debe dominar: elegante, calido, tecnico, editorial, minimalista u otro.

## Proximas decisiones recomendadas

1. Definir objetivo principal del sitio.
2. Definir publico principal.
3. Definir propuesta de valor en una sola frase.
4. Definir direccion visual general.
5. Definir el contenido minimo de Home.
