# Home Wireframe

## Objetivo del documento

Traducir el brief de Home a una estructura visual simple para diseno en GIMP.
El orden esta planteado de arriba hacia abajo.

## Estructura vertical

### 1. Header

- Nombre del bloque: Header
- Proposito: dar contexto inmediato del sitio y permitir acceso rapido a las secciones principales.
- Contenido sugerido:
  - logo o nombre del sitio
  - menu principal con Home, About, Servicios, Blog, Contacto y Tienda
  - boton o enlace visible hacia la accion principal
- Elementos visuales sugeridos:
  - barra superior limpia
  - logotipo simple
  - menu horizontal
  - boton destacado
- Nota breve de diseno: mantenerlo claro y ordenado, con suficiente espacio para que la navegacion se sienta ligera.

### 2. Hero Principal

- Nombre del bloque: Hero Principal
- Proposito: comunicar la propuesta de valor principal en pocos segundos y dirigir al usuario hacia la accion principal.
- Contenido sugerido:
  - titular principal
  - texto breve de apoyo
  - CTA principal: ver servicios
  - CTA secundaria: contactar ahora
- Elementos visuales sugeridos:
  - imagen principal, composicion grafica o mockup visual fuerte
  - fondo distintivo
  - botones bien visibles
- Nota breve de diseno: este bloque debe cargar el mayor peso visual de la pagina y dejar claro de inmediato que ofrece el sitio.

### 3. Introduccion Breve

- Nombre del bloque: Introduccion Breve
- Proposito: explicar de forma corta quien esta detras del sitio o cual es su enfoque general.
- Contenido sugerido:
  - pequeño parrafo de presentacion
  - frase que refuerce confianza o enfoque
- Elementos visuales sugeridos:
  - retrato, imagen de apoyo o recurso grafico secundario
  - composicion simple de texto e imagen
- Nota breve de diseno: debe sentirse mas calmado que el hero y ayudar a humanizar o contextualizar el proyecto.

### 4. Servicios Destacados

- Nombre del bloque: Servicios Destacados
- Proposito: mostrar rapidamente la oferta principal y facilitar que el usuario entienda las opciones disponibles.
- Contenido sugerido:
  - titulo de seccion
  - 3 o 4 tarjetas de servicios
  - nombre breve de cada servicio
  - descripcion corta por tarjeta
  - enlace o boton para ver mas
- Elementos visuales sugeridos:
  - tarjetas con iconos o mini ilustraciones
  - estructura en grid
  - jerarquia clara entre titulo, descripcion y accion
- Nota breve de diseno: conviene que esta seccion sea muy escaneable y que cada tarjeta se entienda de un vistazo.

### 5. Beneficios o Diferenciadores

- Nombre del bloque: Beneficios o Diferenciadores
- Proposito: reforzar confianza y explicar por que esta opcion vale la pena.
- Contenido sugerido:
  - titulo breve
  - lista de 3 o 4 beneficios
  - frases cortas enfocadas en valor o confianza
- Elementos visuales sugeridos:
  - iconos sencillos
  - bloques pequenos o columnas
  - separacion visual respecto a servicios
- Nota breve de diseno: este bloque debe sentirse limpio y directo, sin exceso de texto.

### 6. Blog Destacado

- Nombre del bloque: Blog Destacado
- Proposito: mostrar contenido util y dar una señal de actividad, conocimiento o autoridad.
- Contenido sugerido:
  - titulo de seccion
  - 2 o 3 articulos destacados
  - titulo corto, resumen breve y enlace por articulo
- Elementos visuales sugeridos:
  - tarjetas editoriales
  - miniaturas o imagenes de portada
  - composicion mas ligera que la seccion de servicios
- Nota breve de diseno: debe verse como contenido complementario, no como el foco principal de conversion.

### 7. Vista Previa de Tienda

- Nombre del bloque: Vista Previa de Tienda
- Proposito: mostrar que tambien existe una parte comercial o de productos dentro del sitio.
- Contenido sugerido:
  - titulo de seccion
  - 2 o 4 productos destacados
  - nombre, imagen y referencia breve
  - enlace para entrar a tienda
- Elementos visuales sugeridos:
  - tarjetas de producto
  - imagenes limpias
  - pequena etiqueta de precio o categoria si tiene sentido
- Nota breve de diseno: debe verse atractiva pero secundaria respecto al bloque principal de servicios.

### 8. CTA Final

- Nombre del bloque: CTA Final
- Proposito: cerrar la pagina con una accion clara para usuarios que ya entendieron la propuesta.
- Contenido sugerido:
  - frase de cierre
  - breve texto de apoyo
  - boton principal hacia contacto o servicios
- Elementos visuales sugeridos:
  - bloque ancho y destacado
  - fondo contrastante o tratamiento visual especial
  - boton grande y claro
- Nota breve de diseno: este bloque debe sentirse como cierre intencional y no como un relleno al final.

### 9. Footer

- Nombre del bloque: Footer
- Proposito: cerrar la pagina con navegacion secundaria e informacion basica.
- Contenido sugerido:
  - logo o nombre del sitio
  - enlaces rapidos
  - datos basicos de contacto
  - redes sociales si aplican
- Elementos visuales sugeridos:
  - estructura en columnas simples
  - texto pequeno pero legible
  - separacion clara respecto al CTA final
- Nota breve de diseno: mantenerlo funcional y sobrio, sin competir visualmente con los bloques principales.
